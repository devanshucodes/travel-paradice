### Run Locally

To run **Travel paradice** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/devanshucodes/travel-paradice.git
```

Windows:

```bash
git clone https://github.com/devanshucodes/travel-paradice.git
```
